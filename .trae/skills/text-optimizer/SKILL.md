# 文字优化 Skill

## 功能概述

本 skill 对已撰写的小说内容进行精细化文字优化，包括语句润色、节奏调整、爽点强化、描写优化、对话改进等，提升内容的可读性和吸引力。

## 触发条件

- 用户要求优化小说文字
- 内容审计后需要修改优化
- 需要提升章节质量
- 工作流中审计完成后自动触发

## 优化维度

### 1. AI去痕优化

#### 机器感消除
- **对话自然化**: 将AI生成的生硬对话优化为符合人物身份的自然对话
- **去除AI痕迹**: 消除AI典型的表达方式（如过度优美的辞藻、复杂的句式）
- **口语化处理**: 将书面语转换为更自然的口语表达
- **个性化强化**: 突出每个人物的独特说话方式

#### 语句优化
- **长句拆分**: 将过长的句子拆分为短句
- **语序调整**: 调整语序使表达更自然
- **重复消除**: 删除重复的词语和句式
- **连贯性增强**: 增加过渡词，使句子衔接更顺畅

#### 精简优化
- **冗余删除**: 删除无意义的修饰语
- **精简表达**: 用更少的字表达同样的意思
- **去水词**: 删除"然后"、"接着"等口语化水词
- **AI典型词替换**: 替换AI常用的陈词滥调，如"从此幸福快乐"、"冰山一角"等

### 2. 节奏优化

#### 段落节奏
- **段落长短交替**: 避免连续长段落
- **对话分段**: 长对话拆分为多段
- **空行优化**: 合理使用空行控制阅读节奏

#### 句子节奏
- **短句为主**: 每句不超过30字，符合网文快节奏
- **长短搭配**: 短句制造紧张感，长句用于描写
- **标点节奏**: 用逗号、句号控制阅读节奏
- **爆点节奏**: 关键爽点处使用短句密集排列，增强冲击力

### 3. 描写优化

#### 动作描写优化
```
优化前: 他打了他一拳
优化后: 他一拳砸出，拳风呼啸
```

```
优化前: 他跑得很快
优化后: 他身形如电，瞬间掠过十丈距离
```

#### 场景描写优化
```
优化前: 天气很热
优化后: 烈日当空，大地被烤得发烫，空气中弥漫着灼热的气息
```

```
优化前: 房间很乱
优化后: 衣服散落在地，书本堆满了桌子，空气中飘着灰尘
```

#### 心理描写优化
- **外化心理**: 用动作/表情展现心理，减少直接描写
- **暗示手法**: 通过环境烘托心理
- **精简心理**: 心理描写不超过100字

### 4. 对话优化

#### 个性化对话
```
优化前:
"你好。"他说。
"你好。"她说。

优化后:
"哟，这不是林大天才吗？"他挑了挑眉。
她冷哼一声："少来这套，找我什么事？"
```

#### 对话张力
```
优化前:
"你打不过我的。"
"我会试试的。"

优化后:
"就凭你？"他嗤笑一声，"也配站在我面前？"
主角缓缓握紧拳头，指节发白："试试便知。"
```

#### 对话节奏
- **短句对话**: 紧张场景用短句对话
- **潜台词**: 对话中暗含冲突
- **动作穿插**: 对话中穿插动作描写

## 优化级别

### 1. 轻度优化
- 语句流畅度调整
- 错别字修正
- 标点优化

### 2. 中度优化
- 段落节奏调整
- 对话优化
- 描写增强

### 3. 重度优化
- 爽点强化
- 悬念重写
- 节奏重构

### 4. 定制优化
- 针对特定问题优化
- 根据审计报告优化
- 根据读者反馈优化

```

skill_view(text-optimizer) --focus 爽点强化 --chapters 1-10
```

## 优化原则

1. **保持原意**: 不改变情节走向和人物设定
2. **风格一致**: 优化后风格与原文一致
3. **不过度**: 优化要有度，避免华丽空洞
4. **读者导向**: 以提升读者体验为核心
5. **可追溯**: 记录每次优化内容
